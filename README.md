# Cinema System

Frontend application for managing a cinema movie catalog.

## Features
- Public movie catalog
- Authentication with Clerk
- Protected dashboard
- Movie search with deferred rendering
- Simulated server-side filtering
- Headless DataGrid component
- Compound pattern
- Chat support panel

## Tech Stack
- React
- TypeScript
- Vite
- React Router
- Clerk
- Context API

## Architecture
Explain components, contexts, hooks, services, types, views.

## How to Run
npm install
npm run dev

## Environment Variables
VITE_CLERK_PUBLISHABLE_KEY=your_key_here
